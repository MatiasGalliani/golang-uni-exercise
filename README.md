Hello! Welcome to my first ever functioning project on GOLANG.

For a better understanding I'm going to copy and paste the exercise I had to do:

### Assignment 39

The program processes the following information: **Name**, **Gender**, **Civil Status**, and **Age**.

**Input Codes:**
*   **Gender:** `1 = Male`, `2 = Female`
*   **Civil Status:** `1 = Single`, `2 = Separated`, `3 = Married`, `4 = Widowed`

**Objectives:**
The system must calculate and display:
*   The total count of **married men** older than 30.
*   The total count of **single women** younger than 25.
*   The **average age** of married men.
*   The total number of **men** and **women** entered.
*   The total count of **single men**.
*   The **name** of the oldest woman.

Used official documentation of GOLANG and Google Gemini for help, but the whole syntax was handwritten.
